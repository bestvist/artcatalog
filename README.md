> Artdir is a tool for generating article catalogs.

# Props

| 属性     | 说明           | 类型   |
| -------- | -------------- | ------ |
| el       | 内容元素       | string |
| target   | 目录的存放元素 | string |
| position | 目录的定位     | object |

position options

| 属性   | 说明     | 类型   |
| ------ | -------- | ------ |
| top    | 顶部定位 | string |
| bottom | 底部定位 | string |
| left   | 左侧定位 | object |
| right  | 右侧定位 | object |
